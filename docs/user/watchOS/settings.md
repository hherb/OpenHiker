# Settings

Swipe to the **Settings** tab to configure OpenHiker on your watch.

## GPS Settings

### Accuracy Mode
Choose between three GPS accuracy levels:

- **High** — Best accuracy (5m), highest battery use
- **Balanced** — Good accuracy (10m), moderate battery use (recommended)
- **Low Power** — Basic accuracy (100m), longest battery life

See [Battery Management](battery-management.md) for detailed comparisons.

## Units

### System Preference
- **Metric** — Distances in km, elevations in m
- **Imperial** — Distances in mi, elevations in ft

This preference applies to all displays: map stats, hike statistics, navigation distances, and the stats dashboard.

## Display Settings

### Show Scale
Toggle the map scale bar on or off. The scale bar helps you judge distances on the map.

### Show UV Index
Toggle the UV index badge on or off. When enabled, a color-coded badge appears in the bottom-right of the map showing the current UV exposure level.

## Navigation Settings

### Minimalist Nav
When enabled, starting route navigation opens the battery-saving [Minimalist Navigation](minimalist-nav.md) view instead of the full map.

> "Shows only turn directions to save battery"

## Health Settings

### HealthKit Authorization
Shows current authorization status:
- **Authorized** — Health data is available
- **Not Authorized** — Tap **Authorize** to grant permission

### Record Workouts
Toggle whether GPS recordings are also saved as Apple Health workouts. When enabled:
- Hikes appear in the Apple Health app
- Extended background runtime improves GPS accuracy
- Heart rate monitoring frequency increases

### Error Display
If there's a HealthKit error, it's displayed here with details.

## App Info

- **Version** — Current app version (e.g., 1.0.0)
