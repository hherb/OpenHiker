# OpenHiker for Apple Watch — User Manual

OpenHiker for Apple Watch is a standalone wrist-based hiking navigator with offline topographic maps, GPS tracking, turn-by-turn navigation, and health monitoring.

## Tab Navigation

The watch app uses vertical swipe navigation with 6 tabs:

| Swipe Position | Tab | Purpose |
|----------------|-----|---------|
| Top | Stats Dashboard | Live health and hike statistics |
| Center | **Map** | Primary map view (default) |
| Below map | Routes | Planned routes for navigation |
| Below routes | Regions | Downloaded offline map regions |
| Below regions | Settings | GPS, units, display, health options |
| Bottom | Minimalist Nav | Battery-saving navigation mode |

Swipe up/down to move between tabs.

## Requirements

- Apple Watch Series 6 or later (recommended for SpO2)
- watchOS 10.0 or later
- iPhone with OpenHiker installed (for initial map transfer)
- GPS (built into all Apple Watch models)
