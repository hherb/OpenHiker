# Community Routes

Browse, download, and share hiking routes with the OpenHiker community.

## Browsing Routes

The **Community** tab loads routes from the OpenHikerRoutes GitHub repository.

### Searching and Filtering

- **Search bar** — Filter by route name or text
- **Activity filter chips** — All, Hiking, Cycling
- **Country filter dropdown** — Filter by country
- **Sort options**:
  - Newest first (date)
  - Longest first (distance)
  - Highest first (elevation)
  - Name (A-Z)

### Pull to Refresh
Swipe down to reload the latest routes from the repository.

### Route Cards

Each route shows:
- Activity icon (hiking/cycling)
- Route name and country
- Author name and area
- Brief summary
- Distance, elevation, estimated duration
- Photo count and waypoint count badges

## Route Detail

Tap a route to see:
- Author information
- Full description
- Map preview with route polyline
- Complete statistics
- Photo gallery
- **Download** button to save locally

## Downloading for Offline Use

Tap **Download** to save a community route to your planned routes. You can then:
- View it in the Routes tab
- Start navigation with it
- Review it offline

## Sharing Your Routes

### Uploading a Hike

1. Go to the **Hikes** tab
2. Open a recorded hike
3. Tap **Share** or navigate to Community upload
4. Fill in the upload form:
   - **Description** — Tell other hikers about the route
   - **Difficulty rating** — Rate the difficulty
   - **Tips and recommendations** — Share advice
   - **Photos** — Attach your best trail photos
5. **Preview** your submission
6. **Submit** — Creates a GitHub pull request automatically
