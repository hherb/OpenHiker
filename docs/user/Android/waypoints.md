# Waypoints

Mark and manage points of interest on the trail — viewpoints, water sources, campsites, and more.

## Adding a Waypoint

1. From the **Waypoints** tab, tap the **+** floating action button
2. Or create a waypoint from within a hike or map context

### Waypoint Form

| Field | Description |
|-------|-------------|
| **Photo** | Take with camera or pick from gallery |
| **Category** | Choose from 9 categories (chip group) |
| **Name** | Descriptive label |
| **Notes** | Multi-line description |
| **Coordinates** | Auto-filled from GPS (editable) |

### Categories

| Category | Use For |
|----------|---------|
| Viewpoint | Scenic overlooks, panoramas |
| Water | Streams, springs, water taps |
| Shelter | Shelters, huts, emergency refuges |
| Summit | Mountain peaks, high points |
| Campsite | Camping areas, bivouac spots |
| Danger | Hazardous sections, cliff edges |
| Info | Information boards, trail signs |
| Parking | Trailhead parking areas |
| Custom | Anything else |

## Viewing Waypoints

The **Waypoints** tab shows all your waypoints across all hikes.

### Filtering by Category

Category filter chips appear at the top. Tap one or more to filter:
- Select individual categories to focus on specific types
- Tap **Clear** to reset all filters

### Waypoint Cards

Each waypoint shows:
- Thumbnail photo or category icon
- Name
- Category badge with color coding
- Distance from your current GPS location

### Detail View

Tap a waypoint to see:
- Full-resolution photo (zoomable)
- Name and category
- GPS coordinates with **copy button**
- Notes
- Associated hike (tap to navigate to hike detail)
- Mini-map with waypoint pin
- Edit and delete options

## Managing Waypoints

- **Swipe to delete** — With confirmation dialog
- **Edit** — From the detail view
- Waypoints are stored locally and sync via cloud folder (if configured)
