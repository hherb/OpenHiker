# Getting Started with OpenHiker for Android

## Installation

Install OpenHiker from the Google Play Store or build from source.

## Permissions

On first launch, OpenHiker requests the following permissions:

| Permission | Why | Required? |
|-----------|-----|-----------|
| **Location** | GPS tracking and map centering | Yes |
| **Background location** | Continue tracking when screen is off | Yes (for hike recording) |
| **Camera** | Taking photos for waypoints | Optional |
| **Storage/Photos** | Attaching gallery photos to waypoints | Optional |
| **Notifications** | Foreground service notification during hike recording | Yes (Android 13+) |

Grant at least location permission to use the app. Other permissions can be granted later when needed.

## First Launch

1. **Map screen** — The app opens to the Navigate tab showing an interactive online map
2. **Bottom navigation bar** — Five tabs: Navigate, Regions, Hikes, Routes, Community
3. **Settings** — Gear icon in the top-right corner

## What to Do First

1. **Browse the map** — Explore using pinch-to-zoom and drag
2. **Download a region** — Tap the download button to save the visible area for offline use
3. **Plan a route** — Go to Routes tab to create a hiking or cycling route
4. **Or browse community routes** — Go to Community tab for inspiration

## Adaptive Layout

OpenHiker adapts to your screen size:

- **Phones** (compact width) — Bottom navigation bar with 5 tabs
- **Tablets and foldables** (medium/expanded width) — Navigation rail on the left side

The app supports both portrait and landscape orientations.
