## App Navigation

OpenHiker uses a bottom navigation bar (phones) or navigation rail (tablets):

| Tab | Purpose |
|-----|---------|
| **Navigate** | Interactive map with offline tiles and GPS |
| **Regions** | Download and manage offline map regions |
| **Hikes** | View recorded hike history |
| **Routes** | Plan routes and manage planned routes |
| **Community** | Browse and share community routes |

A **Settings icon** in the top app bar is accessible from all screens.

## Requirements

- Android 8.0 (API 26) or later
- GPS-enabled device
- Internet connection for downloading maps (offline after download)
- Camera (optional, for waypoint photos)

## Key Differences from iOS

| Feature | Android | iOS |
|---------|---------|-----|
| Map engine | MapLibre | MapKit + SpriteKit |
| Cloud sync | Google Drive / Dropbox / any SAF provider | iCloud |
| Health data | Not available | Apple Watch HealthKit |
| Watch support | Not available | Apple Watch app |
| P2P sharing | Not available | MultipeerConnectivity |
| Route import | Community routes | Community + GPX + Apple Maps |
