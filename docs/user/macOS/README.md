## App Layout

OpenHiker for Mac uses a sidebar navigation layout:

```
┌─────────────┬──────────────────────────────┐
│  Sidebar     │  Main Content Area           │
│             │                              │
│  Regions    │  (selected section's view)   │
│  Downloaded │                              │
│  Hikes      │                              │
│  Waypoints  │                              │
│  Routes     │                              │
│  Community  │                              │
│             │                              │
│  ─────────  │                              │
│  iCloud Sync│                              │
└─────────────┴──────────────────────────────┘
```

The sidebar lists all sections. The main content area shows the selected section. An iCloud sync status bar sits at the bottom of the sidebar.

## Requirements

- macOS 14.0 (Sonoma) or later
- Internet connection for downloading maps (offline after download)
- iCloud account (optional, for syncing with iPhone and Apple Watch)

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Cmd + I | Import GPX file |
| Cmd + Shift + S | Sync with iCloud |
| Cmd + Shift + P | Send routes to iPhone (via iCloud) |
| Cmd + , | Open Settings |
