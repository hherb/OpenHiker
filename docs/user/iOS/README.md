# OpenHiker for iOS — User Manual

OpenHiker for iPhone and iPad is the full-featured companion app for downloading maps, planning routes, recording hikes, and managing your Apple Watch.

4. [Navigation](navigation.md) — Turn-by-turn guidance and live GPS tracking
5. [Recording Hikes](recording-hikes.md) — GPS tracking, stats, and saving your adventures
6. [Waypoints](waypoints.md) — Marking and managing points of interest
7. [Apple Watch Transfer](watch-transfer.md) — Sending maps and routes to your watch
8. [Community Routes](community-routes.md) — Browsing and sharing routes with other hikers

10. [Exporting Data](exporting-data.md) — GPX, PDF reports, and markdown summaries
11. [Settings & Preferences](settings.md) — Units, map styles, and cloud sync

## Platform Overview

| Feature | iPhone | iPad |
|---------|--------|------|
| Map download | Yes | Yes |
| Route planning | Yes | Yes |
| Turn-by-turn navigation | Yes | Yes |
| GPS track recording | Yes | Yes |
| Apple Watch transfer | Yes | Yes |
| Peer-to-peer sharing | Yes | Yes |
| Community routes | Yes | Yes |
| Sidebar navigation | — | Yes |
| Waypoint list tab | — | Yes |

iPad uses a sidebar layout with all features accessible from a NavigationSplitView, while iPhone uses a tab bar.

## Requirements

- iOS 17.0 or later
- iPhone or iPad with GPS
- Internet connection for downloading maps (offline after download)
- Apple Watch (optional, for watch features)
